Styles:

There are multiple styles that can be used:

The text can be in different sizes.

The font can be different.

The text can be aligned differently.

The background-color can be different.

The text can be in different colors.

The text can have a colored border.

Text formating:

The text can be formated in multiple different ways:
Bold text. No extra importance.
Strong text. This defines text of extra importance.
This is used for technical terms, words from other languages, etc.
This is used to emphasize text.
Small text.
Marked/highlighted text.
"Deleted" text.
"Inserted" text. (Underlined).
Subscripted text and superscripted text.

Title

The title attribute enables you to show some text when you hover the mouse over whatever element you are creating.

Horizontal rules:

To separate content or define a change, the "hr"-element can be used to create a horizontal line, like the one above this heading.
